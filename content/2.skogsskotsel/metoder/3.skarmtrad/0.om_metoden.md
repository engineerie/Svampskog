Överhållen skärm innebär att antalet lämnade skyddande skärmträd är fler än vid en vanlig fröträdsställning och att de sparas längre än vid normalt skärmskogsbruk. Skärmträden avvecklas successivt i flera steg medan trädplantor etableras och växer till.

::icon-bullet{icon="i-streamline-tree-3" }
Tall och bok fungerar bra att föryngra under överhållen skärm. Gran är vindkänsligare och lämpar sig därför sämre.
::
::icon-bullet{icon="i-heroicons-sun" }
Överhållen skärm kan vara ett bra val i skogar med naturvärden som vill ha ljus och värme.
::
::UAlert{variant="subtle" color="tertiary" title="För att räknas som en hyggesfri metod enligt Skogsstyrelsens definition måste minst 25 träd per hektar lämnas att ingå i det nya beståndet." class="my-6 "}
::
